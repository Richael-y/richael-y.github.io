漏洞分析

漏洞位于文件 WEB-INF/classes/com/fh/controller/common/Uploader.class

路由是 /common/getDrive.do

前面两个 if判断都是在 type 为 library 的时候进入，我们这里传入的是 type=drives ，直接进入到第三个 if语句；

调用 File.listFiles ，直接列出对应文件下的文件，然后放入到 map 中返回。
